# pdf-sign
extract pdf signe date
* Usage:
    ```
    pdf-sign <path_to_pdf>
    ```
* debug
    ```
    cargo build
    target/debug/pdf-sign <path_to_pdf>
    ```
    or
    ```
    cargo run <path_to_pdf>
    ```
* release
    ```
    cargo build --release
    target/release/pdf-sign <path_to_pdf>
    ```
